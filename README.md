# Just Looking: Shadow Archer

Ein kleines erstes Godot-4-PC-Spielprojekt für Windows/Linux/macOS.

## Spielidee

Du spielst einen Bogenschützen in einer dunklen Arena. Gegner kommen in Wellen auf dich zu. Du weichst aus, zielst mit der Maus und schießt Pfeile. Jeder besiegte Gegner gibt Score und Coins. Nach jeder Welle wird der Spieler automatisch etwas stärker.

## Steuerung

- **WASD**: Bewegen
- **Maus**: Zielen
- **Linksklick**: Schießen
- **ESC**: Pause
- **SPACE**: Spiel starten
- **R**: Nach Game Over neu starten

## Lokal testen

1. Godot 4 installieren.
2. Dieses Repository herunterladen oder klonen.
3. Godot öffnen.
4. **Import** wählen.
5. Die Datei `project.godot` auswählen.
6. Projekt öffnen und oben rechts auf **Play** drücken.

## Aktueller Stand

Dies ist ein erster spielbarer Prototyp mit:

- Startscreen
- Spielerbewegung
- Maus-Zielen
- Pfeilen
- Gegnerwellen
- Schaden und Leben
- Score, Coins und Wave-Anzeige
- Game Over und Restart
